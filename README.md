## Getting Started

### Docker Deployment

To build and run using Docker:

```bash
docker build -t portfolio .

# Run the container
docker run -p 3000:3000 portfolio
```

### Installation & Deployment Via Node Package Manager

Install the dependencies & Deploy:

```bash
npm install
npm run dev
```

The website will be available at `http://localhost:5173`.
